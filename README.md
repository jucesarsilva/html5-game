# html5-game

A simple game with html5 (canvas)

## Demo
[Here](https://jucesarsilva.github.io/html5-game/index.html)

### Dependecies
```
npm install --global gulp-cli
```

### Config
> it's will install 
```
yarn
```

### Start
> it's will start
```
yarn start
```

### Build
> it's will build a bindle.js
```
gulp
```
