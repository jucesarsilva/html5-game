# html5-game
A simple game

- Html5 (canvas)
- Javascript ES6 with class inheritance
- Modules
- Scss global and module support
- Webpack mode dev && mode prod

## Demo
[Here](https://jucesarsilva.github.io/html5-game/index.html)


### Config
> it's will install 
```
yarn
```

### Start
> it's will start
```
yarn start
```

### Build
> it's will build a bindle.js
```
yarn build
```
